===============
 tpms test apk 
===============

Описание

APK модуль тестового контроля давления в шинах, на платформе Android, для системы tpms контроля давления в шинах.
Работает через последовательный порт UART подключенный к модулю ТРMS; Реализовано программное обеспечение c протоколом связи последовательного порта. 

Код протокола связи в apk реализация слоя средой jni, серийный порт по умолчанию /dev/ttyS0, скорость передачи информации в бодах 9600.
На Android, необходимо убедиться, что для устройство /dev/ttyS0 есть права доступа, модуль TPMS имеет нормальный источник питания, и вы можете использовать этот АПК для тестирования.

Использование инструкций

1. apk в status - Отображает состояние соединения аппаратного модуля
2. refresh all - Кнопка может обновить все состояние (включая шину и предупреждение)
3. tire* - Дает в общей сложности пять данных о статусе шин
4. match - Кнопка для сопоставления (обучения) шин
5. macth cancel - Кнопка отмены процесса сопоставления шин
6. unwatch - Кнопка используется для удаления шины
7. match all - Сопоставить все шины
8. unwatch all -  Удалить все шины
9. alert* - Выдает в общей сложности шесть конфигураций аварийных сигналов
10.config и config all - Используется для настройки аварийных сигналов
